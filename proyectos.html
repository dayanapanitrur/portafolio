SELECCIONADOS

Una colección de interfaces, sistemas visuales y exploraciones cromáticas desarrolladas en diferentes ecosistemas digitales.

Fecha MARZO 2024
Cliente / Collab STUDIO NÓMADA
Identidad / Web

Cromática Agua

Desarrollo de una identidad visual basada en la refracción de luz y superficies líquidas. El objetivo fue crear una marca que se sintiera viva, utilizando el espacio negativo y transparencias dinámicas.

Fecha OCTUBRE 2023
Cliente / Collab TECH LABS INC.
UI / UX Design

Ecosistema Azul

Diseño de interfaz para una aplicación móvil con micro-interacciones que simulan la física de fluidos. Se priorizó una paleta monocromática de azules cobalto para generar profundidad sin sobrecargar cognitivamente al usuario.

Fecha ENERO 2023
Cliente / Collab PROYECTO PERSONAL
Dirección de Arte

Gravedad Cero

Exploración visual sobre la ausencia de peso en entornos digitales. Uso intensivo de cromados, grises neutros y tipografía helvética distorsionada para generar una estética futurista pero elegante.
